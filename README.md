# Pokemon Application

The Pokemon Application uses The RESTful PokÃ©mon API available on https://pokeapi.co/ site.

## Input 

The application supports input type `attacker -> defender`

For example:
```
fire -> grass
fighting -> ice rock
psychic -> poison dark
water -> normal
fire -> rock
```

Example Ouput:
```
2.0x
1.0x
```
